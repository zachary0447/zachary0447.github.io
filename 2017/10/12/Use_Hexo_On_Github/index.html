如何使用 Hexo 在 GitHub 上搭建博客

写博客对学习有很多好处。写博客的过程能让你更清晰地审视自己的学习经历，弄清每一个细节；同时，还能锻炼你的语言表达能力；随时记录自己的收获，还能获得成就感。

什么是 Hexo？

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

Hexo 的安装前提

  • Node.js
  • Git
