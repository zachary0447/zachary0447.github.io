使用 CSS 画太极图

用 CSS 绘制太极图。是 CSS tricks 上某个小技巧的修改。
众所周知，太极图是这样的。

经过思考，我们可以把太极图分成三个部分：

  1. 一个大的圆形，左半边是黑色的，右半边是白色的。
  2. 过大圆形的圆心 O ，做一根垂直于屏幕的垂线 AB。以 OA 为直径做一个黑色的大圆，一个白色的小圆。
  3. 以 OB 为直径，做一个白色的大圆，黑色的小圆。
    总的来说，一共五个圆。为了制作方便，可以分成三部分来做。恰好是一个 div，和这个 div 的:before 和:after。

第一部分

第一个大圆我们使用 border 和 div 本身的互动来实现。
首先绘制一个 width: 48px;height: 96px; 的 div，然后设置为黑色，这就是黑色的左半边。
之后绘制 border-width: 2px 50px 2px 2px; 的边框，设置为白色，这就是白色的右半边。
现在是个方形，再加上 border-radius: 100%; 就是一个左黑右白的圆了。

第二部分和第三部分

第二部分和第三部分是非常相似的。首先是将 div 设置为 position: relative;，然后将 before 和 after 设置为 position: absolute;，即可通过绝对定位找到圆心。
在top: 0; left: 50%;的地方绘制一个白色的圆，加上黑色的边框，就形成了太极的上半部分。
下半部分是top: 50%; left: 50%;，黑色圆，白色边框。其他代码是非常相似的。
